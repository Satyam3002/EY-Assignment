# Admin Dashboard UI

A React-based Admin Dashboard built from Figma design using Vite and Tailwind CSS.

## Tech Stack

- React 19
- Vite
- Tailwind CSS v4

## Setup

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

3. Open http://localhost:5173

## Build

```bash
npm run build
```

## Components

- **Header** - Top navigation with search, dark mode toggle, notifications
- **Sidebar** - Collapsible navigation menu with dark/light mode support
- **Footer** - Page footer with links
- **TransactionsCard** - Financial transactions list
- **UpgradePlanCard** - Subscription upgrade section
- **MeetingScheduleCard** - Upcoming meetings display

## Features

- Dark/Light mode toggle for sidebar
- Expandable search bar
- Collapsible dashboard submenu
- Responsive design
- Pixel-perfect Figma implementation
